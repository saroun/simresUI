Three axis thermal neutron spectrometer (TAS).
TAS configuration with input slit in primary beam and doubly focusing analyzer and monochromator.

Remarks:

Use tracing options double cycle=true and storage mode=sample in order to simulate resolution functions properly.

SAMPLE Generic component is used for simulations of resolution functions.
Resolution profile 1D Use this command to view simulated resolution profile along selected directions.
Resolution map 2D Plots resolution function projected on a give plane, represented as a grayscale map.
